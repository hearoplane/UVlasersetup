Microscope Configuration
========================

Configuring a new microscope involves subclassing a BaseMicroscopeGUI and creating an associated user interface (ui file).

Current Configured Microscopes are:
 * 1213 Laser Lab Micoscope 1: TRPLMicroscopeGUI
 * 1210 Supra CL Microscope: CLMicroscopeGUI
